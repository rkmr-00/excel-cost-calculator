# Excel Cost Calculator — 原価計算マクロ

Excelで品目・数量・単価を入力するだけで、小計・合計の集計と棒グラフを自動生成するVBAマクロです。

## 機能

- 品目・数量・単価を入力 → ワンクリックで集計シートを自動生成
- 各品目の小計・総合計を計算
- 品目別コスト棒グラフを自動作成
- セットアップマクロで入力シートのひな形を即作成

## 使い方

### 1. マクロを取り込む

1. Excelを開き、`Alt + F11` でVBAエディタを起動
2. 左ペインのモジュールを右クリック →「ファイルのインポート」
3. `CostCalculator.bas` を選択してインポート

### 2. 入力シートを準備する

VBAエディタまたはマクロ一覧から `SetupInputSheet` を実行すると、  
サンプルデータ入りの「入力」シートが自動作成されます。

### 3. データを入力する

「入力」シートに以下の形式でデータを入力します：

| 品目     | 数量 | 単価（円） |
|----------|------|------------|
| 材料費A  | 10   | 5,000      |
| 外注費B  | 2    | 30,000     |
| 経費C    | 1    | 8,000      |

### 4. 集計を実行する

`CalculateCost` マクロを実行すると：
- 「集計」シートに小計・合計が出力される
- 品目別コストの棒グラフが自動生成される

## 動作環境

- Microsoft Excel 2016 以降（Windows）
- マクロ有効ブック（.xlsm）で使用してください

## ファイル構成

```
excel-cost-calculator/
├── CostCalculator.bas   # VBAモジュール（メイン）
```

## カスタマイズ

- 品目数は無制限（A列が空白になるまで自動検出）
- グラフの種類は `xlBarClustered` を変更することで変更可能
- 色設定は `RGB()` の値を変更してカスタマイズ可能
